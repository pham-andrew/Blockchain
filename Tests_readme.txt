TESTS READ ME
This document describes the tests in the smart city model system.

The tests show that we can...
1. Add Cities: add multiple cities to the controller.
2. Add Devices: add devices to a city.
3. Add People: add people to a city.
4. Events: update events on a device
5. Smart City Sample: run the script provided by the class.
6. Smart City Test: run the script provided by the class with extra show outputs at the end to check.
7. Update Device: update devices.
8. Compile: Compile and run the project as described in the gradesheet.
